실습 준비

Overleaf 프로젝트 세팅

1

계정 생성

overleaf.com → Register → 무료 계정으로 시작

2

새 프로젝트 생성

New Project → Blank Project → 이름: "PM25_Thesis_이름"

3

docs/draft 내용 붙여넣기

methods.md, results.md, discussion.md 내용을 main.tex의 각 \section에 옮겨 넣습니다.

LaTeX 최소 문법

오늘 필요한 문법만 5줄

\section{연구방법} % docs/draft/methods.md 내용
\section{결과} % docs/draft/results.md 내용
\section{고찰} % docs/draft/discussion.md 내용
PM$_{2.5}$   % 아래첨자
$\mu$g/m$^3$  % 단위 표현
