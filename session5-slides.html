📄 개요
Overleaf란?
Overleaf = 브라우저에서 바로 쓰는 LaTeX 논문 편집기
설치 없이 · 공동 편집 · 수천 개 저널 템플릿 · 자동 컴파일
🌐
설치 불필요
브라우저만 있으면 OK. Mac·Windows·어디서나 동일한 환경
👥
실시간 공동편집
지도교수·공동저자와 동시에 편집. 변경 이력 자동 저장
📑
저널 템플릿
Elsevier, AGU, AMS 등 수천 개 공식 템플릿 무료 제공
Word vs LaTeX/Overleaf
Word는 수식·참고문헌 관리가 힘듦. LaTeX는 수식·인용이 자동화되고 저널 포맷이 완벽하게 맞음
무료 계획으로도 충분
무료 계정으로 저장소 생성·공동편집(1명)·컴파일 모두 가능

🛠 실습 1
Overleaf 프로젝트 시작
1
overleaf.com 접속 → Register (학교 이메일 권장)
2
New Project → Blank Project 또는 From Template 선택
3
템플릿 검색: 투고 목표 저널명 입력 (예: Atmospheric Environment)
4
좌측 파일 목록에서 main.tex 클릭 → 편집 시작
latex — 기본 문서 구조
\documentclass{article}
\usepackage[utf8]{inputenc}
\usepackage{natbib}  % 참고문헌 패키지

\title{서울시 PM2.5 장기 트렌드 분석}
\author{홍길동}

\begin{document}
\maketitle

\section{Introduction}
본 연구는 ... \cite{WHO2021} ...

\bibliographystyle{apalike}
\bibliography{references}  % references.bib 파일 연결
\end{document}

📚 파일 관리
references.bib 관리 요령
bibtex — references.bib 구조 예시
%% ── 대기질 관련 ──────────────────────
@article{WHO2021guidelines,
  title   = {WHO global air quality guidelines},
  author  = {{World Health Organization}},
  year    = {2021},
  url     = {https://www.who.int/...}
}

@article{Park2023seoul,
  title   = {Seasonal variation of PM2.5 in Seoul},
  author  = {Park, S. and Lee, K.},
  journal = {Environ. Sci. Technol.},
  volume  = {57}, pages = {1234--1245},
  year    = {2023}
}

%% ── 방법론 ───────────────────────────
@book{Shumway2017,
  title  = {Time Series Analysis and Its Applications},
  author = {Shumway, R.H. and Stoffer, D.S.},
  year   = {2017}
}

Key 이름 규칙 통일: 저자성+연도+키워드 (예: Kim2022seoul) · 주제별 주석으로 섹션 구분
06 / 16
✍️ 실습
본문에 \cite{} 삽입하기
latex — 다양한 인용 방식
% 기본 인용 — (Kim, 2022)
PM2.5는 WHO 기준을 초과하였다 \cite{Kim2022seoul}.

% 저자 이름 포함 — Kim (2022)에 따르면
\citet{Kim2022seoul}에 따르면 서울의 PM2.5는...

% 여러 문헌 동시 인용
여러 연구에서 감소 추세가 확인되었다 \cite{Kim2022seoul, Park2023seoul}.

% 페이지 지정
\cite[p.~15]{WHO2021guidelines}
참고문헌 스타일 변경
\bibliographystyle{apalike} APA
\bibliographystyle{ieeetr} IEEE
\bibliographystyle{nature} Nature
\bibliographystyle{agu} AGU
자동 정렬·번호
인용 추가/삭제 시 참고문헌 목록과 번호가 자동으로 업데이트됩니다.

🔎 레퍼런스 수집
Google Scholar 외 BibTeX 수집 방법
출처BibTeX 찾는 방법
Google Scholar논문 하단 " → BibTeX 클릭
DOI 직접 입력doi2bib.org 에서 DOI 입력 → BibTeX 생성
PubMed논문 페이지 → Cite → Download .nbib → 변환
Elsevier / ScienceDirect논문 페이지 → Export → BibTeX
AMS / AGU 저널논문 페이지 → Tools → Citation → BibTeX
arXiv논문 페이지 → Export BibTeX 링크
💡 가장 빠른 방법
DOI만 있으면 doi2bib.org 에서 바로 완성된 BibTeX 생성. DOI는 보통 논문 첫 페이지에 있습니다.
