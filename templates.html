## Decision Rules

- A: 4개 이상이 Yes이고 내 연구질문과 직접 연결된다.
- B: 일부 기준만 맞지만 배경이나 방법 설명에 쓸 수 있다.
- C: 용어, 데이터 출처, 넓은 맥락 확인에만 쓴다.
- Exclude: 핵심 변수, 데이터, 방법, 연구문제가 내 주제와 다르다.

## How to Use

- Introduction의 모든 주장에 최소 1개 이상의 근거 논문을 연결한다.
- 근거가 약한 문장은 표현을 낮추거나 삭제한다.
- "아무도 하지 않았다"처럼 강한 주장은 매우 조심해서 쓴다.

Reproducibility Checklist

# Reproducibility Checklist

- [ ] 데이터 출처가 명시되어 있는가?
- [ ] 원본 데이터와 전처리 데이터가 구분되어 있는가?
- [ ] 전처리 코드가 있는가?
- [ ] 분석 코드가 순서대로 실행되는가?
- [ ] `requirements.txt`가 있는가?
- [ ] Figure 생성 코드가 있는가?
- [ ] 논문에 사용된 수치가 코드 결과와 일치하는가?
- [ ] README만 읽고 실행할 수 있는가?
