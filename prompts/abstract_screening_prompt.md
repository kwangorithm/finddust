# Abstract Screening Prompt

아래 논문 제목과 초록을 보고 내 연구에 읽을 가치가 있는지 스크리닝해줘.

내 연구질문:
[여기에 연구질문 작성]

판단 기준:
1. 연구질문과 직접 관련이 있는가?
2. PM2.5, PM10, air quality 등 핵심 변수가 맞는가?
3. 데이터 지역과 기간이 명확한가?
4. 방법론이 내 연구와 비교 가능한가?
5. 정량 결과나 평가 지표가 있는가?
6. 내 논문에서 어디에 쓸 수 있는가?

출력:
- Grade: A / B / C / Exclude
- 이유:
- 내 논문에서 쓸 위치:
- 원문에서 반드시 확인할 항목:

중요:
- 초록에 없는 내용은 추측하지 마.
- 애매하면 "원문 확인 필요"라고 표시해.

논문 제목과 초록:
[여기에 붙여넣기]
