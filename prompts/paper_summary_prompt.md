# 논문 요약 프롬프트

```text
다음 논문 초록과 방법론 일부를 읽고 아래 형식으로 요약해줘.

1. 연구 문제
2. 사용 데이터
3. 입력 변수
4. 예측/분석 대상
5. 사용 모델
6. 비교 모델
7. 성능 지표
8. 핵심 결과
9. 저자가 주장하는 contribution
10. 내가 반드시 원문에서 다시 확인해야 할 항목

중요:
- 논문에 없는 내용은 추측하지 마.
- 모르는 정보는 "원문 확인 필요"라고 표시해.
- 성능 수치, 데이터 기간, 지역명은 반드시 따로 표시해.

논문 내용:
[여기에 초록/방법/결과 일부 붙여넣기]
```
