2. 사용 데이터

- 성능 수치, 데이터 기간, 지역명은 반드시 따로 표시해.

3. 데이터 지역과 기간이 명확한가?

6. 내 논문에서 어디에 쓸 수 있는가?

- Grade: A / B / C / Exclude
- 이유:
- 내 논문에서 쓸 위치:

- 초록에 없는 내용은 추측하지 마.
