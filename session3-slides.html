⌨️ 실습 1
미세먼지 데이터 전처리 코드 생성
💬 Claude Code에 입력할 프롬프트
data/raw/ 폴더에 AirKorea에서 받은 서울 PM2.5 CSV 파일들이 있어. 파일 구조를 확인하고, 다음 작업을 하는 src/preprocess.py를 만들어줘:
1) 여러 연도 파일을 하나로 합치기
2) 결측치(-999, NaN) 처리
3) 날짜 컬럼을 datetime으로 변환
4) data/processed/seoul_pm25_clean.csv로 저장
python — Claude Code가 생성하는 코드 예시
import pandas as pd
import glob
from pathlib import Path

def load_and_clean(raw_dir: str) -> pd.DataFrame:
    files = glob.glob(f"{raw_dir}/*.csv")
    dfs = [pd.read_csv(f, encoding='cp949') for f in files]
    df = pd.concat(dfs, ignore_index=True)

    # 결측치 처리
    df.replace(-999, pd.NA, inplace=True)
    df['pm25'] = pd.to_numeric(df['pm25'], errors='coerce')

    # 날짜 변환
    df['datetime'] = pd.to_datetime(df['일시'])
    return df.dropna(subset=['pm25'])

⌨️ 실습 2
통계 분석 코드 생성
💬 프롬프트
data/processed/seoul_pm25_clean.csv를 분석해서 src/analysis.py를 만들어줘:
1) 연도별 · 월별 평균, 최대, 초과일수(WHO 기준 15μg/m³) 통계표
2) Mann-Kendall 트렌드 검정으로 장기 감소 추세 유의성 확인
3) 결과를 results/stats_summary.csv로 저장
python — 생성 예시
import pandas as pd
import pymannkendall as mk

df = pd.read_csv('data/processed/seoul_pm25_clean.csv',
                 parse_dates=['datetime'])

# 연도별 통계
yearly = df.groupby(df['datetime'].dt.year)['pm25'].agg(
    mean='mean', max='max',
    exceed_who=lambda x: (x > 15).sum()
).reset_index()

# 트렌드 검정
result = mk.original_test(yearly['mean'])
print(f"트렌드: {result.trend}, p={result.p:.4f}")
💡 팁
pymannkendall 같은 패키지를 몰라도 돼요. Claude Code가 알아서 적절한 라이브러리를 선택하고 설치까지 해줍니다.

⌨️ 실습 3
논문용 시각화 코드 생성
💬 프롬프트
results/stats_summary.csv를 사용해서 논문 Figure 1을 만드는 src/visualize.py를 작성해줘. 연도별 PM2.5 연평균 바차트에 WHO 기준(15)과 한국 기준(15) 수평 점선 추가. 학술지 제출 기준(300dpi, 단컬럼 너비 88mm)에 맞게 저장해줘.
python — 논문 품질 Figure 생성 예시
import matplotlib.pyplot as plt
import matplotlib as mpl

mpl.rcParams['font.family'] = 'NanumGothic'
fig, ax = plt.subplots(figsize=(3.46, 2.6))  # 88mm

ax.bar(yearly['year'], yearly['mean'], color='#4fc3f7', alpha=0.8)
ax.axhline(15, color='red', ls='--', lw=0.8, label='WHO 기준')
ax.set_xlabel('연도'); ax.set_ylabel('PM2.5 (μg/m³)')

fig.savefig('figures/fig1_pm25_trend.tiff',
            dpi=300, bbox_inches='tight')

# 미세먼지 연구 프로젝트 맥락

## 연구 개요
서울시 PM2.5 장기 트렌드 분석 (2015–2023)
AirKorea 시간별 데이터 사용

## 데이터 규칙
- 원본 데이터: data/raw/ (수정 금지)
- 전처리 후: data/processed/
- 결측치 코드: -999
- 인코딩: CP949 (한국 CSV)

## 코드 규칙
- Python 3.11, pandas 2.x
- 시각화: matplotlib (한글폰트: NanumGothic)
- 논문 Figure: 300dpi, TIFF 형식
