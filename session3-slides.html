실습 1

PM2.5 분석 코드 첫 번째 생성

📝 CLAUDE CODE 프롬프트 예시
나는 안양대 미세먼지관리전공 대학원생이야.
현재 pm25-thesis/analysis 폴더에 작업 중이고,
AirKorea에서 받은 PM2.5 일별 측정 데이터 (날짜, 측정소명, PM2.5 농도)를 분석하려고 해.

다음 기능을 하는 Python 스크립트를 analysis/pm25_analysis.py로 만들어줘:
1. CSV 파일 읽기 (pandas)
2. 결측값(-999, NaN) 제거
3. 봄/여름/가을/겨울 계절별 평균 PM2.5 계산
4. 계절별 박스플롯 저장 (figures/seasonal_boxplot.png)
✅ Claude Code는 파일을 직접 생성하고, 코드를 실행해보고, 오류가 나면 스스로 수정합니다.

프롬프트 템플릿

미세먼지관리전공 수업별 프롬프트

🌀
대기확산모델링 이론
"AERMOD 출력 파일(.dat)을 읽어서 지점별 PM2.5 최대 농도 분포를 등고선 지도로 그려줘. x축은 동서 거리(m), y축은 남북 거리(m)"

# Python 실행 중 오류 발생 시
Traceback (most recent call last):
  File "analysis/pm25_analysis.py", line 23, in <module>
    df['date'] = pd.to_datetime(df['date'], format='%Y%m%d')
ValueError: time data '2023/01/01' does not match format '%Y%m%d'

# Claude Code에 그냥 붙여넣기!
> "위 오류가 났어. 원인이 뭔지 설명하고 고쳐줘"

→ 날짜 형식이 '20230101'이 아닌 '2023/01/01'로 되어 있어요.
→ format='%Y/%m/%d' 또는 format='mixed'로 수정하겠습니다.
😌 오류 메시지를 그대로 붙여넣으면 됩니다. "왜 안 돼?"라고 물어도 됩니다.
오류를 무서워하지 마세요 — 이제 AI가 같이 고쳐줍니다.
